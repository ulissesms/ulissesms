# Project Title

## Description

This is a project that demonstrates the use of flexbox CSS to align stack icons and applications side by side.

## Installation

Follow the instructions below to install the project:

1. Clone the repository:
   ```bash
   git clone https://github.com/ulissesms/ulissesms.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ulissesms
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage

To start the application, run:
```bash
npm start
```

## Flexbox Example CSS

```css
.stack-icons {
    display: flex;
    justify-content: space-around;
}

.application {
    margin: 10px;
}
```
